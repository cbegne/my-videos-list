# My-video-list

## Synopsis

A list system to manage videos you want to watch (external youtube links)

## Motivation

Try out:
- Chakra-UI,
- json-server,
- react-hook-form

## Which features are done

List of videos to watch:
- add, delete, update,
- see videos details,
- simple navigation,
- simple search by id

## Which features are not done yet

- Responsiveness

## Languages

React

## To start

`npm run dev`
