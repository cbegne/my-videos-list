# My-video-list

## Synopsis

A list system to manage videos you want to watch (external youtube links)

## Motivation

Try out:
- chakra-UI,
- json-server,
- react-hook-form.

## Which features are done

List of videos to watch:
- add, delete, update,
- see videos details,
- simple navigation,
- simple search by id.

## Which features are not done yet

- responsiveness

## Languages

React
